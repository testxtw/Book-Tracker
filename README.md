# Book-Tracker
GUI-приложение «Book Tracker» для управления списком прочитанных книг с возможностью фильтрации.

**Автор:** Рахимова Виолетта Андреевна

## Описание программы

Book Tracker — приложение для учёта прочитанных книг с возможностью добавления новых книг, фильтрации по жанру и количеству страниц, а также сохранения данных в файл JSON.

## Инструкция по запуску

1. Установите Python 3.x.
2. Установите зависимости:
   pip install -r requirements.txt (если требуется)
3. Запустите приложение:
   python book_tracker.py

## Примеры использования

- Добавьте книгу через форму.
- Используйте фильтры для поиска по жанру или количеству страниц.
- Данные сохраняются автоматически в books.json.
