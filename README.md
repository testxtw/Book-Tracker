# Book-Tracker
GUI-приложение «Book Tracker» для управления списком прочитанных книг с возможностью фильтрации.

**Автор:** Рахимова Виолетта Андреевна

Описание: Простое приложение для учёта прочитанных книг. Позволяет добавлять книги с полями: название, автор, жанр и количество страниц. Есть фильтрация по жанру и количеству страниц. Все данные сохраняются в файл books.json.Зависимости: Только стандартная библиотека Python (tkinter).Имя файла с кодом: book_tracker.py.Структура проекта:

Копировать
Book-Tracker/
│   book_tracker.py   # Главный файл приложения.
│   books.json         # Файл данных.
│   README.md          # Эта документация.
Инструкция по запуску:

Установите Python 3.x.
Скачайте или клонируйте репозиторий.
Запустите файл: python book_tracker.py.
Используйте интерфейс для добавления и фильтрации книг.Примеры использования:
Добавьте книгу «Война и мир», автор «Лев Толстой», жанр «Роман», 1225 страниц.
Примените фильтр по жанру «Роман» или по количеству страниц > 1000.
Все изменения сохраняются автоматически.
